## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- Git e Github
- Figma
- Visual Studio Code

## 💻 Projeto

Este é um projeto de Pricing Table que oferece uma maneira fácil e visualmente atraente de apresentar diferentes opções de preços para seus produtos ou serviços.
